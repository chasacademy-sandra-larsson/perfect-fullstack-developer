## KURSENS HUVUDSAKLIGA INNEHÅLL

Kursen fördjupar den studerandes förståelse för fullstackutveckling genom arbete med moderna tekniker och ramverk. Fokus ligger på cloud-baserad drift, CI/CD-pipelines och DevOps-metoder för automatiserad och kvalitetssäkrad leverans av webbtjänster. Studerande introduceras till molnbaserade hostinglösningar (t.ex. AWS, Azure, Google Cloud eller motsvarande), serverless-arkitektur, avancerad versionshantering med Git samt automatiserade CI/CD-flöden.

I fullstack-fördjupningen behandlas GraphQL som alternativ till REST-API, och den studerande får arbeta med ett valbart JavaScript-ramverk utöver React — exempelvis Vue, Svelte eller Angular — för att bredda förståelsen av ramverksmönster. Inom valt ramverk behandlas komponentstruktur, routing, state management, rendering, prestanda och skalbarhet. Headless CMS introduceras som integrationspartner för moderna fullstackapplikationer.

AI-verktyg används som stöd i utvecklingsarbetet — t.ex. för att tolka dokumentation, generera tester eller felanalysera CI/CD-flöden — med fortsatt kritisk granskning av output. Innehåll som tidigare täckts (NoSQL, Docker, testning, OWASP-säkerhet, TypeScript som fullstack-språk) repeteras inte utan antas som förkunskap.

## KURSMÅL

Vid avslutad kurs med godkända resultat, kan den studerande:

**KUNSKAPER:**

1. Redogöra för cloud hostinglösningar (t.ex. AWS, Azure, Google Cloud eller motsvarande) samt serverless-arkitektur
2. Redogöra för CI/CD pipelines och DevOps-metoder för kvalitetssäkrad leverans
3. Redogöra för avancerad versionshantering med Git
4. Redogöra för GraphQL och dess användningsområden i förhållande till REST
5. Redogöra för grundläggande tekniker och uppbyggnad inom ett valbart JavaScript-ramverk (t.ex. Vue, Svelte eller Angular)
6. Redogöra för routing och state management inom valt ramverk samt andra verktyg inom ramverkets ekosystem
7. Redogöra för rendering, prestanda och skalbarhet inom valt ramverk
8. Redogöra för headless CMS och dess integration med moderna fullstackapplikationer

**FÄRDIGHETER:**

9. Implementera och driftsätta webbtjänster på cloud hostinglösningar
10. Tillämpa avancerad funktionalitet i versionshantering med Git
11. Tillämpa CI/CD pipeline för automatiserad leverans
12. Planera och implementera ett GraphQL-API
13. Utveckla en webbtjänst i valt JavaScript-ramverk med routing och state management
14. Integrera och arbeta med headless CMS i ett fullstackprojekt

**KOMPETENSER:**

15. Argumentera för val av API, databas, säkerhet och hostingstrategi i ett fullstackprojekt
16. Motivera och välja lösningar för routing, state management, struktur, skalbarhet och prestanda inom valt JavaScript-ramverk

## PRINCIPER FÖR BETYGSSÄTTNING

Betyg sätts i form av icke godkänt (IG), godkänt (G) eller väl godkänt (VG)

**Icke godkänt (IG):** Samtliga kursmål är inte uppfyllda

**Godkänt (G):** Samtliga kursmål är uppfyllda

**Väl godkänt (VG):** Den studerande har genomfört kursen och uppnått godkänt i alla kursmål. Dessutom kan den studerande med stor säkerhet och skicklighet:

- Argumentera för val av API, databas, säkerhet och hostingstrategi i ett fullstackprojekt (kursmål 15)
- Motivera och välja lösningar för routing, state management, struktur, skalbarhet och prestanda inom valt JavaScript-ramverk (kursmål 16)
