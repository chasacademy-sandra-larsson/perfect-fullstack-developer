## KURSENS HUVUDSAKLIGA INNEHÅLL

Kursen behandlar grundläggande webbteknik och webbutveckling med fokus på HTML, CSS och moderna JavaScript-standarder (ES2024+), samt versionshantering av kod med Git. Kursen ger förståelse för webbens uppbyggnad och HTTP-kommunikation (request/response), semantisk och tillgänglig HTML, modern CSS med responsiv design samt JavaScript för interaktiv funktionalitet och DOM-manipulation. De studerande arbetar med browser-API:er, hämtar och hanterar data från externa API:er med Fetch och JSON, lagrar data lokalt i webbläsaren med localStorage samt följer grundläggande säkerhetspraxis i klient-kod. Principer inom UX/UI, användbarhet och tillgänglighet (a11y/WCAG) behandlas som en yrkesmässig baslinje, och studenten övar på att ta fram mockups och wireframes som underlag för utveckling samt implementera webbgränssnitt utifrån designskisser. Genomgående används AI-verktyg för att effektivisera utvecklingsarbetet där det är möjligt, med ett kritiskt förhållningssätt till genererat innehåll.

## KURSMÅL

Vid avslutad kurs med godkända resultat, kan den studerande:

**KUNSKAPER:**

1. Redogöra för principer inom UX/UI, användbarhet och tillgänglighet
2. Redogöra för HTML som semantisk struktur för webbsidor
3. Redogöra för modern CSS (box model, Grid, Flexbox) samt responsiv design
4. Redogöra för moderna JavaScript-standarder (ES2024+) — variabler, datatyper, kontrollflöde, funktioner, scope, arbete med arrayer och objekt, samt språkfeatures som spread/rest, destructuring och modulsystem
5. Redogöra för DOM:en, händelsehantering och hur webbsidor manipuleras dynamiskt
6. Redogöra för browser-API:er för datahämtning och lagring (Fetch, localStorage) samt dataformatet JSON
7. Redogöra för HTTP-protokollet och request/response i webbkommunikation

**FÄRDIGHETER:**

8. Ta fram mockups och wireframes för webbgränssnitt
9. Utveckla responsiva och tillgängliga webbsidor med semantisk HTML och modern CSS
10. Genomföra utveckling av webbgränssnitt utifrån en given designskiss
11. Utveckla funktionalitet med JavaScript enligt moderna standarder (ES2024+)

**KOMPETENSER:**

12. Självständigt driva och utveckla en responsiv, tillgänglig webbplats med HTML, CSS och JavaScript
13. Självständigt driva grundläggande versionshantering av kod genom exempelvis Git

## PRINCIPER FÖR BETYGSSÄTTNING

Betyg sätts i form av icke godkänt (IG), godkänt (G) eller väl godkänt (VG)

**Icke godkänt (IG):** Samtliga kursmål är inte uppfyllda

**Godkänt (G):** Samtliga kursmål är uppfyllda

**Väl godkänt (VG):** Den studerande har genomfört kursen och uppnått godkänt i alla kursmål. Dessutom kan den studerande med stor säkerhet och skicklighet:

- Självständigt driva och utveckla en responsiv, tillgänglig webbplats med HTML, CSS och JavaScript (kursmål 12)
